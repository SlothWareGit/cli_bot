# cli_bot
This is a simple CLI bot I want to continously work on as I learn python.

## Commands
    "help": "shows this message",
    "clear": "clears the console",
    "say": "makes the bot say something (usage: say [message])",
    "generate": "generates a random sha256 encryption",
    "refresh": "refreshes the program to apply code changes",
    "pswrd-gen": "generates a random password with letters, digits, and special characters",
    "exit": "exits the program"
